# HealthCare-Dashboard---Power-BI

## 📋 Project Description
This Power BI project provides a comprehensive analysis of a healthcare center's financial performance and provider efficiency. The goal was to transform raw data into a structured, interactive dashboard that helps stakeholders identify key performance indicators (KPIs) and significant healthcare trends.

## 📈 Dashboard Requirements & Deliverables
The project consists of a **3-page interactive dashboard** designed to meet the following requirements:
* **Financial Overview:** A summary of the healthcare center's financial health.
* **Provider Insights:** An analysis of provider performance, efficiency, and workload.
* **Trend Analysis:** Identification of significant trends and patterns over time.
* **Additional Insights:** Deep-dive analysis into specific metrics derived from the dataset.

## 🛠️ Technical Implementation
* **Data Cleaning:** Used **Power Query** to handle missing values, format data types, and prepare the dataset for analysis.
* **DAX Modeling:** Developed custom measures using **DAX** to calculate KPIs, profit margins, and performance ratios.
* **Visualization:** Applied best practices in data visualization using **Bar Charts, Line Charts, Gauges, and Slicers** for maximum interactivity.
* **Design:** Ensured the dashboard is user-friendly with professional formatting (100% - 400% zoom compatibility).

## 📂 Project Files
* `Healthcare_Dashboard.pbix`: The main Power BI report file.
* `Healthcare_Dataset.xlsx`: The raw dataset used for the analysis.
* `Screenshots/`: A folder containing images of each dashboard page.

## 🚀 How to Run the Project
1.  Download the repository.
2.  Open the `.pbix` file using **Power BI Desktop**.
3.  Navigate through the 3 pages using the navigation buttons/tabs.
